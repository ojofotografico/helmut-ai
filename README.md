---

## 🏛️ Arquitectura del Ecosistema (Modular System)

Siguiendo estándares de ingeniería de alto rendimiento, el repositorio se organiza de forma desacoplada para garantizar el blindaje de la Propiedad Intelectual (IP):

- **`.assets/`**: Gestión de recursos visuales bajo protocolo de compresión extrema (<50KB) sin pérdida de fidelidad.
- **`.docs/`**: Repositorio confidencial de procedimientos operativos (SOPs), estrategias de pauta digital y manuales de infraestructura.
- **`.src/`**: Núcleo del sistema. Incluye el motor bilingüe dinámico y la lógica de validación perimetral de leads.
- **`.tools/`**: Biblioteca táctica de **Helmut AI Academy** (Engineering Prompts & Data Literacy).

---

## 🛠️ Pilares de Ingeniería

### ⚡ Rendimiento (High-Performance)
- **Zero-Dependency Engine:** Desarrollo en Vanilla JavaScript para eliminar latencia.
- **Visuals:** Interfaz basada en *Glassmorphism Pro* (Blur 24px, Saturate 180%) con tipografía fluida adaptativa.
- **SEO Ready:** Optimización nativa para Google Core Web Vitals (LCP < 1.2s).

### 🧠 Inteligencia de Conversión
- **Multi-language Core:** Motor dinámico bilingüe (i18n) con persistencia de estado.
- **Diagnostic Briefing:** Módulo de contacto con sistema de pre-auditoría técnica que califica leads en tiempo real antes de la ingesta de datos.
