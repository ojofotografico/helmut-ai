# HELMUT AI — Precision Engineering

Proyecto de vanguardia centrado en consultoría de Inteligencia Artificial, presencia digital y academia técnica.

## 🛠️ Pilares Técnicos
- **Velocidad:** Vanilla JavaScript (Zero dependencies), CSS Nativo, SVGs inyectados.
- **Imagen:** Soporte bilingüe (i18n) y sistema dinámico de temas (Dark/Light).
- **Diseño:** Glassmorphism Pro (Blur 24px, Saturate 180%), Tipografía Fluida.

## 📂 Estructura del Proyecto
- `index.html`: Estructura semántica y metadatos SEO.
- `css/main.css`: Design Tokens, variables y estilos maestros.
- `js/theme.js`: Motor de gestión de temas y persistencia.
- `js/i18n.js`: Diccionario y motor de traducción dinámico.
- `js/main.js`: Inicializador de eventos y lógica global.

## 🚀 Despliegue Local
1. Clonar el repositorio.
2. Abrir `index.html` en un navegador o usar VS Code Live Server.# helmut-ai
